Proxy Constructor

new Proxy(target,handler)

target => Object / function /url which we want to proxy

handler => callback function

A proxy in js can be used to modify the fundamental operations of the an object.A Proxy on an object is itself an object,that has traps

Traps may be triggered when opearations are performed on the proxy.(propert lookup[get],function calling,modifing properties,adding properties[set],removing properties[delet])
